# Proposed System Diagrams - Labsych

This directory contains Mermaid diagrams representing the **proposed Labsych digital platform** for laboratory equipment booking and management.

## System Overview

**Labsych** is a web-based platform that will replace the current manual system with:
- **Digital inventory management** replacing physical logbooks
- **Automated booking system** replacing phone calls
- **Integrated payment processing** with M-Pesa
- **Real-time availability** visible to all schools
- **Maintenance tracking** and damage management
- **Analytics and reporting** for data-driven decisions

## Transformation from Current to Proposed System

| Current System | Proposed Labsych System |
|---------------|------------------------|
| Phone calls for requests | Web-based browsing & booking |
| Manual logbook checking | Real-time database queries |
| Verbal confirmations | Automated email/SMS notifications |
| Paper receipts | Digital transaction records |
| No damage tracking | Photo-based damage documentation |
| Manual calculations | Automated pricing algorithm |
| No analytics | Comprehensive reporting dashboard |
